플러그인

플러그인 작성

플러그인은 일반적으로 전역 수준 기능을 Vue에 추가합니다. 플러그인에는 엄격하게 정의된 범위는 없습니다. 일반적으로 작성할 수있는 플러그인에는 여러 유형이 있습니다.

  1. 약간의 전역 메소드 또는 속성 추가 예. vue-element

  2. 하나 이상의 글로벌 에셋 추가 : 지시문 / 필터 / 전환 등. vue-touch

  3. 글로벌 mixin으로 일부 컴포넌트 옵션을 추가하십시오. 예. vuex

  4. Vue.prototype에 Vue 인스턴스 메소드를 연결하여 Vue 인스턴스 메소드를 추가하십시오.

  5. 가지고 있는 API를 제공하면서 동시에 위의 일부 조합을 주입하는 라이브러리. 예. vue-router

Vue.js 플러그인은install 메소드를 노출해야합니다. 이 메소드는 첫 번째 인자로 Vue 생성자와 함께 가능한 옵션과 함께 호출 될 것입니다.

플러그인 사용하기

Vue.use() 글로벌 메소드를 호출하여 플러그인을 사용하십시오.

선택적으로 몇 가지 옵션을 전달할 수 있습니다.

Vue.use는 자동으로 동일한 플러그인을 두 번 이상 사용하지 못하기 때문에 동일한 플러그인에서 여러 번 호출하면 플러그인이 한 번만 설치됩니다.

vue-router와 같은 Vue.js 공식 플러그인이 제공하는 일부 플러그인은Vue가 전역 변수로 사용 가능한 경우 자동으로Vue.use()를 호출합니다. 그러나 CommonJS와 같은 모듈 환경에서는 항상Vue.use()를 명시 적으로 호출해야합니다 :

awesome-vue에서 커뮤니티에서 거대한 기여한 플러그인 및 라이브러리 컬렉션을 확인하세요
